This api contains information for the NBA Eastern Conference.

There are 15 different and separate sections which include an Id,teams,shopping,and roster.

Inside teams there are 4 parts which are the team name, team image, team record, and team conference.

Inside of shopping there is 3 parts which are Jersey, Description, and Price.

Inside of roster there are 13 different sections which include 3 elements inside of each section. In each section there is a playerID, player, and image.
